Simple web-based calculator

Intended focus:

  • Usability — Large, well visible buttons (particularly good when used on a mobile device). Convenient layout.
  • Maintainability — MVC separation of the content/layout (HTML), design (CSS), and logic (JavaScript).
  • Compatibility — To work on all contemporary browsers, including the mobile device ones.
  • Standard — Well-formed HTML5. JS code in own namespace and follow nowadays coding syntax and rules.

Notes:

  • This development is in regard to a test task, part of my application for Frontend Developer at Eyeo GmbH.
  • As the task asked for "exponentiation" — this could be achieved by x**y, e.g. 2**2
  • As required, no external JavaScript libraries are used. Everything is done from scratch.
